# Stema da Shopee
> É uma pequena aplicação em NEST.JS que integra com monorepo para termos os microserviços em um mesmo repostitório

## Arquitetura de Microserviços
> Esta arquitetura serve para gerenciarmos as funcionalidades em vários outros microserviços para que possamos distribuir as reponsabilidades para os outros micro serviços
